# foundry-agent-network-diagnostic
# ---------------------------------------------------------------------------
# The core diagnostic (src/diagnose.py + the 6 checks + HTML/JSON report)
# runs on the Python 3.10+ STANDARD LIBRARY ONLY. No third-party packages are
# required, so it works inside closed/air-gapped networks with no PyPI access.
#
# Azure queries are performed by shelling out to the Azure CLI (`az`), which you
# authenticate separately with `az login`. There is nothing to pip-install for
# the diagnostic itself.
# ---------------------------------------------------------------------------
